# Investigación: GitHub Projects

## 1. ¿Qué son los GitHub Projects?

Son tableros de gestión de tareas integrados en GitHub que permiten organizar issues, PRs y notas en columnas como "To Do", "In Progress" y "Done". Ayudan a planificar y hacer seguimiento del progreso del proyecto sin salir de GitHub.

## 2. GitHub Projects vs Kanban tradicional

- **Kanban tradicional**: tablero físico o en herramientas externas como Trello o Jira, desconectado del código
- **GitHub Projects**: integrado directamente con issues y PRs del repositorio, los cambios en el código se reflejan automáticamente en el tablero

## 3. Automatización en GitHub Projects

GitHub Projects permite automatizar tareas mediante Workflows:
- Mover un issue a "Done" automáticamente cuando se cierra
- Agregar issues nuevos automáticamente a la columna "Todo"
- Mover un PR a "In Progress" cuando se abre para revisión
